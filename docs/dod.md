# LobsterAI 阶段交付 DoD（Definition of Done）

## 1. 适用范围

- 适用于当前核心功能阶段。
- 覆盖代码、联调、文档、回归、可回滚性。

## 2. 功能完成标准

1. 需求在 `roadmap-v1.md` 或对应迭代条目中有明确映射。
2. 关键路径可端到端演示通过（至少 1 条正向链路 + 1 条异常链路）。
3. 无阻断级已知缺陷遗留。

## 3. 质量门禁

1. `npm run lint` 通过。
2. `npm run compile:electron` 通过。
3. 回归集通过（models/chat/stream/auth/scheduler）。
4. 动画联动回归通过（五态事件与回退策略）。

## 4. 联调验收标准

1. OpenAI 兼容接口可用：
   - `GET /v1/models` 正常返回
   - `POST /v1/chat/completions` 正常返回
2. 流式输出完整：
   - 不出现短回复截断
3. 会话清理正确：
   - 不出现异常会话堆积
4. AIRI 动画联动：
   - 五态事件触发成功率 >= 99%

## 5. 文档完成标准

1. 变更已记录到对应文档：
   - `roadmap-v1.md`
   - `milestones.md`
   - `animation-link-protocol-v1.md`（涉及动画时）
2. 风险与回滚方案已更新。
3. 本次验证步骤可被他人复现。

## 6. 观测与运维标准

1. 关键错误有清晰日志，不以告警掩盖阻断问题。
2. 重要状态变化（会话、任务、动画事件）可追踪。
3. 已识别噪音日志有降级或过滤策略。

## 7. 回滚标准

1. 可在 10 分钟内回滚至上一个稳定版本。
2. 回滚后主链路可恢复。
3. 回滚步骤文档化并可执行。

## 8. 范围冻结约束

1. 本阶段不新增 IM 平台接入能力。
2. 本阶段不推进手机 IM 触发桌面 Agent 新需求。
3. IM 仅做稳定性维护，不作为本阶段新增交付项。
