## 任务列表分类（文件夹功能）更新说明

### 功能概述

用户可以将侧边栏中的任务（会话）归类到自定义文件夹中，方便在任务数量较多时进行分组管理。文件夹名称持久化存储于本地 SQLite 数据库，重启后不丢失。

### 数据流

```
用户右键任务 → 选择"移动到文件夹" → 输入/选择文件夹名
  → coworkService.setSessionFolder()
  → IPC: cowork:session:setFolder
  → main/coworkStore.setSessionFolder()  ← UPDATE cowork_sessions SET folder = ?
  → dispatch updateSessionFolder()       ← Redux 同步更新
  → CoworkSessionList 重新分组渲染
```
