# mindmap（思维导图/白板）

标签：`mindmap`

## 基本用法

```html
<mindmap type="mindmap" snapshot="https://example.com/snapshot.png"></mindmap>
```

## 属性

| 属性            | 值           | 说明                     |
| --------------- | ------------ | ------------------------ |
| `type`          | `mindmap` 等 | 类型，默认 `mindmap`     |
| `snapshot`      | URL          | 快照图片地址（可选）     |
| `snapshotimage` | URL          | 快照图片备选地址（可选） |

## 说明

- 该标签同时承载 mindmap 和 whiteboard
