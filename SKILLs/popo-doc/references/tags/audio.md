# audio（音频）

标签：`audio`

## 基本用法

```html
<audio source="https://example.com/audio.mp3" title="语音"></audio>
```

## 属性

| 属性      | 值                   | 说明             |
| --------- | -------------------- | ---------------- |
| `source`  | URL                  | 音频地址         |
| `title`   | 字符串               | 音频标题（可选） |
| `loading` | `"true"` / `"false"` | 加载状态（可选） |

# 说明

- 暂不支持 AI 上传文件
