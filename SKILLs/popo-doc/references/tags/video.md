# video（视频）

标签：`video`

## 基本用法

```html
<video src="https://example.com/video.mp4" title="演示视频"></video>
```

## 属性

| 属性      | 值                   | 说明             |
| --------- | -------------------- | ---------------- |
| `src`     | URL                  | 视频地址         |
| `title`   | 字符串               | 视频标题（可选） |
| `size`    | 数字                 | 文件大小（可选） |
| `mode`    | 字符串               | 显示模式（可选） |
| `loading` | `"true"` / `"false"` | 加载状态（可选） |

# 说明

暂不支持 AI 上传视频文件
