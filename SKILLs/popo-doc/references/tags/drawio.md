# drawio（Draw.io 画图）

标签：`drawio`

## 基本用法

```html
<drawio url="https://example.com/draw" referencekey="abc" version="2"></drawio>
```

## 属性

| 属性           | 值     | 说明           |
| -------------- | ------ | -------------- |
| `url`          | URL    | 画图地址       |
| `referencekey` | 字符串 | 引用键（可选） |
| `version`      | 数字   | 版本号（可选） |
